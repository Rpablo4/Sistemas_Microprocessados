Projeto desenvolvido em linguagem C para o PIC16F874A e simulado no Proteus Professional.

-Componentes eletrônicos usados:
  Transistor PNP;
  Resistor 100Ohms;
  Diodo Sil;
  Relay 5v;
  Buzzer 5v;
  Fonte 6v;
  2x Push Button;
  Chave SWSPDT;
  Teclado Matricial 4x4;
  LCD 16x2;
  Microcontrolador PIC16F874A;
  
Obs: Projeto suscetível a mudanças e adaptações caso seja colocado em pratica. 
Recomenda-se o uso de componentes SMD, e versões menores do teclado matricial e LCD.

Relógio usado como referência:
CASIO VINTAGE DBC-32-1A
https://www.casio-intl.com/br/pt/wat/watch_detail/DBC-32-1A/
